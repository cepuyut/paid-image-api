How it works

Built for AI agents. No signup, no API keys.

1
POST your prompt
Send to /v1/images/generate with model "auto" for smart routing.
2
Receive 402 challenge
Server responds with amount, recipient, and token address.
3
Pay USDC on Tempo
Transfer USDC to the server wallet. Gas fees covered.
4
Get your image
Retry with payment credential. Cached for instant replay.

Quick start

Works with any HTTP client.

curl
Node.js
Python
# Step 1: Get payment challenge
curl -X POST https://paid-image-api.onrender.com/v1/images/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "a sunset over mountains", "model": "auto"}'

# Step 2: Pay via Tempo, then retry with credential
curl -X POST https://paid-image-api.onrender.com/v1/images/generate \
  -H "Content-Type: application/json" \
  -H "Authorization: Payment <base64url-credential>" \
  -d '{"prompt": "a sunset over mountains", "model": "auto"}'
const res = await fetch("https://paid-image-api.onrender.com/v1/images/generate", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    prompt: "a sunset over mountains",
    model: "auto"
  })
});

// Parse 402 challenge, pay via Tempo SDK, retry
import requests

res = requests.post(
    "https://paid-image-api.onrender.com/v1/images/generate",
    json={"prompt": "a sunset over mountains", "model": "auto"}
)
