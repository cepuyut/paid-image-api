# PixelPay — Agent Instructions

> AI image & video generation API. Pay-per-request via MPP on Tempo blockchain. No API keys needed.

## Quick Start

1. POST to https://pixelpayapi.com/v1/images/generate with {"prompt": "your prompt"}
2. You will receive HTTP 402 with a payment challenge in WWW-Authenticate header
3. Pay the required USDC amount on Tempo blockchain to the specified recipient
4. Retry the same request with Authorization: Payment <credential>
5. Receive 200 OK with your generated image

## Model Selection Guide

- Need it cheap and fast? Use `fal-ai/flux/schnell` ($0.029)
- Budget-friendly alternative? Use `xai/grok-imagine-image` ($0.039)
- Need creative/artistic output? Use `fal-ai/gpt-image-1/text-to-image` ($0.069)
- Need text in the image? Use `fal-ai/ideogram/v3` ($0.079)
- Need best quality? Use `fal-ai/nano-banana-pro` ($0.19)
- Need illustrations/vectors? Use `fal-ai/recraft-v3` ($0.059)
- Need to edit an existing image? Use POST /v1/images/edit ($0.079)
- Need video? Use POST /v1/videos/generate ($0.35)
- Unsure? Omit model field — auto-selects best model based on prompt

## Example: Generate Image

```
POST /v1/images/generate
Content-Type: application/json

{
  "prompt": "a serene mountain landscape at sunset",
  "model": "fal-ai/flux/schnell",
  "image_size": "landscape_4_3"
}
```

## Example: Edit Image

```
POST /v1/images/edit
Content-Type: application/json

{
  "prompt": "replace the sky with a dramatic sunset",
  "image_url": "https://example.com/photo.jpg",
  "mask_url": "https://example.com/sky-mask.png"
}
```

## Example: Generate Video

```
POST /v1/videos/generate
Content-Type: application/json

{
  "prompt": "a cat walking through a garden, cinematic slow motion",
  "image_url": "https://example.com/cat.jpg"
}
```

## Request Parameters (Image Generate)

| Parameter | Type | Required | Description |
|---|---|---|---|
| prompt | string | Yes | Text describing the image |
| model | string | No | Model ID (default: fal-ai/flux/schnell) |
| image_size | string | No | landscape_4_3, square_hd, portrait_4_3, etc. |
| num_images | integer | No | 1-4 images per request (default: 1) |
| style | string | No | anime, cinematic, vintage, noir, cyberpunk, etc. |
| enhance | boolean | No | Auto-enhance prompt for better quality |
| negative_prompt | string | No | Things to avoid |
| seed | integer | No | For reproducible results |
| image_urls | array | No | Reference images for img2img |

## Other Endpoints

- GET /v1/prices — Current pricing for all models
- GET /v1/models — List all available models with capabilities
- GET /v1/styles — Available style presets
- POST /v1/validate — Validate prompt before paying
- POST /v1/images/upscale — Upscale image 2x
- GET /v1/gallery — Public image gallery
- GET /health — Health check

## Payment Details

- Chain: Tempo (chain ID 4217)
- Currency: USDC (0x20c000000000000000000000b9537d11c60e8b50)
- Recipient: 0xf48c6E46358652fC77462d56d609B8bC7f4ba82e
- Fee payer: Server pays gas (feePayer: true)
- Protocol: MPP (Machine Payments Protocol)

## Tips for Agents

1. Always check GET /v1/prices first to budget your spending
2. Use the cheapest model (schnell $0.029 or grok $0.039) for drafts
3. GPT-Image ($0.069) excels at creative/artistic prompts
4. Include style presets to get better results without complex prompts
5. Set enhance=true to auto-improve your prompts
6. Use seed for reproducible results when iterating
7. The server pays gas fees — you only pay the USDC amount
8. For video, provide an image_url as starting frame for better results
