# PixelPay — Agent Instructions

> AI image generation API. Pay-per-request via MPP on Tempo blockchain. No API keys needed.

## Quick Start

1. POST to https://pixelpayapi.com/v1/images/generate with {"prompt": "your prompt"}
2. You will receive HTTP 402 with a payment challenge in WWW-Authenticate header
3. Pay the required USDC amount on Tempo blockchain to the specified recipient
4. Retry the same request with Authorization: Payment <credential>
5. Receive 200 OK with your generated image

## Model Selection Guide

- Need it cheap and fast? Use `fal-ai/flux/schnell` ($0.029)
- Need text in the image? Use `fal-ai/ideogram/v3` ($0.079)
- Need best quality? Use `fal-ai/nano-banana-pro` ($0.19)
- Need illustrations/vectors? Use `fal-ai/recraft-v3` ($0.059)
- Unsure? Omit model field — auto-selects best model based on prompt

## Example Request

```
POST /v1/images/generate
Content-Type: application/json

{
  "prompt": "a serene mountain landscape at sunset",
  "model": "fal-ai/flux/schnell",
  "image_size": "landscape_4_3"
}
```

## Request Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| prompt | string | Yes | Text describing the image |
| model | string | No | Model ID (default: fal-ai/flux/schnell) |
| image_size | string | No | landscape_4_3, square_hd, portrait_4_3, landscape_16_9, portrait_9_16 |
| num_images | integer | No | 1-4 images per request (default: 1) |
| style | string | No | anime, cinematic, vintage, noir, cyberpunk, watercolor, oil-painting, pixel-art, minimalist, pop-art |
| enhance | boolean | No | Auto-enhance prompt for better quality |
| negative_prompt | string | No | Things to avoid (e.g. "blur, watermark") |
| seed | integer | No | For reproducible results |
| image_urls | array | No | Reference images for img2img (Nano Banana, Recraft, Dev only) |

## Other Endpoints

- GET /v1/prices — Current pricing for all models
- GET /v1/models — List all available models with capabilities
- GET /v1/styles — Available style presets
- POST /v1/validate — Validate prompt before paying
- POST /v1/images/upscale — Upscale image 2x ({"image_url": "..."})
- GET /v1/gallery — Public image gallery (JSON)
- GET /health — Health check

## Payment Details

- Chain: Tempo (chain ID 4217)
- Currency: USDC (0x20c000000000000000000000b9537d11c60e8b50)
- Recipient: 0xf48c6E46358652fC77462d56d609B8bC7f4ba82e
- Fee payer: Server pays gas (feePayer: true)
- Protocol: MPP (Machine Payments Protocol)

## Tips for Agents

1. Always check GET /v1/prices first to budget your spending
2. Use the cheapest model (schnell) for drafts, upgrade for final output
3. Include style presets to get better results without complex prompts
4. Set enhance=true to auto-improve your prompts
5. Use seed for reproducible results when iterating
6. The server pays gas fees — you only pay the USDC amount
